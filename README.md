# Cryptocontainer Lab

Минималистичный инструмент для поиска криптографических контейнеров на подключённых дисках и в файловых структурах. Проект ориентирован на быстрый скрининг смонтированных носителей без ведения кейсов или отчётности.

## Возможности
- **Сканирование дисков и каталогов**: поиск криптоконтейнеров на смонтированных дисках (например, `C:\\`, `/mnt/d`) и в отдельных файлах.
- **CLI и GUI**: удобные интерфейсы для запуска сканирования и просмотра найденных совпадений.

## Требования
- Python **3.11+**
- Зависимости из `pyproject.toml`: `typer`, `rich` (для GUI дополнительно `PyQt6`).

## Установка
```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -e .  # установка в режиме разработки
```

## Быстрый старт
Просканируйте подключённый диск или каталог через CLI:
```bash
cryptocontainer-lab scan C:\\
cryptocontainer-lab scan /mnt/d
cryptocontainer-lab scan ./containers
```

## Запуск
- Активируйте виртуальное окружение и установите зависимости (см. раздел «Установка»).
- Проверьте доступные команды: `cryptocontainer-lab --help`.
- Запускайте нужные сценарии как обычные CLI-команды (см. примеры выше). Для GUI установите зависимости `pip install -e .[gui]` и используйте точку входа `python -m cryptocontainer_lab.gui`.

## Графический интерфейс
GUI предназначен только для сканирования дисков:
- **Выбор пути**: укажите диск, каталог или файл и запустите сканирование.
- **Таблица результатов**: найденные сигнатуры контейнеров с типом, путём и смещением.
- **Журнал событий**: сообщения о ходе сканирования и ошибках доступа.

## Функционал и основные команды
Инструмент предоставляет один основной сценарий:
- `scan <path>` — поиск криптографических контейнеров по сигнатурам в файлах на выбранном диске или в каталоге.

## Структура проекта
```
pyproject.toml
src/
  cryptocontainer_lab/
    cli/       # команды CLI, точка входа: main.py
    core/      # базовые модели и константы
    detector/  # логика обнаружения контейнеров
    gui/       # набросок графического интерфейса
```

## Примечания
- Инструмент остаётся прототипом: используется сигнатурный поиск заголовков контейнеров, без криптографической проверки.
- Для работы GUI требуется установка зависимостей из набора `gui`:
  ```bash
  pip install -e .[gui]
  ```
